KIMI Pacemaker v1.0 Design Specification

Target Specifications Summary

Pacing Mode VVI/R (Ventricular demand, Rate-responsive)
Lower Rate Limit 30-100 BPM (programmable)
Upper Rate Limit 100-180 BPM (programmable)
Pacing Amplitude 0-8V (0.25V steps)
Pulse Width 0.1-1.5ms (0.1ms steps)
Sensitivity 0.5-20mV (programmable)
Lead Impedance 200-1500 Ω
Battery Lithium-Iodine (Li/I₂) 1.2Ah
Communication MICS (402-405MHz) + BLE
MRI Compatibility Conditional 1.5T and 3T

Electrical Subsystems

  • Hermetic Encapsulation: Titanium Grade 1 case with laser-welded seam, Glass-to-metal feedthrough
  • Power Management: Li/I₂ battery (2.8V), DC-DC converters, Charge pump (0-8V)
  • Sensing Front End: Instrumentation amplifier (60dB gain), 0.5-50Hz bandpass, 12-bit ADC
  • Main Processor: ARM Cortex-M0+ (48MHz), 256KB Flash, 32KB RAM, 64KB FRAM
  • Pacing Output: H-bridge biphasic output, Programmable amplitude/pulse width

Software Architecture

  • Application Layer: Mode Controller (VVI/R), Rate Response, Capture Management
  • AI/ML Layer: Auto-Threshold Learning, Arrhythmia Detection, Pattern Recognition
  • Safety Features: Watchdog Timer, Power-On Reset, Noise Reversion, Safety Pacing
  • Communication: MICS Band + Bluetooth LE, AES-256 encryption

Power Consumption Budget

Subsystem Active Sleep Duty Cycle Average
Microcontroller 100 µA 0.1 µA 1% 1.1 µA
Sensing AFE 5 µA 0.5 µA 100% 5.0 µA
Pacing Output 5000 µA 0 µA 0.1% 5.0 µA
Telemetry (MICS) 5000 µA 0 µA 0.01% 0.5 µA
Telemetry (BLE) 3000 µA 0 µA 0.1% 3.0 µA
TOTAL ~15 µA

Battery Life: 1.2Ah / 15µA = ~12.5 years
